## Monty's External Crosshair

A lightweight always-on-top crosshair overlay for Windows built with Electron. 
Designed for games that don't have a built-in crosshair, or when you want a 
consistent crosshair across any game or app.

### Features
- 5 crosshair styles — cross, circle, dot, X, and custom image
- Full color customization with color picker and opacity control
- Adjustable size, thickness, gap, outline, and dot
- D-pad position control to align the crosshair to your screen center
- Global hotkey to toggle the crosshair on/off instantly
- Presets for quick setup — default, dot, sniper, classic, circle
- Transparent click-through overlay — your mouse works normally underneath
- Works on top of any game or application

### Built With
- Electron
- HTML / CSS / JavaScript
- electron-builder (for packaging to .exe)
